
# YourTask react/ts
- task management application with login/register.

# I learned:
- How works firebase.
- How to add dark mode.

## Installation

1. Clone this repo

```sh
git clone https://github.com/1991Pawel/react-firebase-yourtask.git
```

2. Install NPM packages

```sh
npm install 
```
3. Start a development server
npm 

```sh
npm run start
```
